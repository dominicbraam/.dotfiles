# .dotfiles

This repo contains most of my personal dotfiles.

Dotfiles are managed with GNU Stow. For help, visit the following link: https://alexpearce.me/2016/02/managing-dotfiles-with-stow/

``` bash
stow -vS $prog-dir
```
